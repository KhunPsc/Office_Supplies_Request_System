# 📦 ระบบแจ้งคำขอซื้อวัสดุสำนักงาน PEA จอมทอง

**Web Application** สำหรับอำนวยความสะดวกในการแจ้งขอซื้อวัสดุสำนักงาน สำหรับพนักงาน PEA แผนกต่างๆ ในสังกัดจอมทอง

## ✨ ฟีเจอร์หลัก
- 📝 **Request Form:** ออกแบบให้ส่งคำของ่าย พร้อมระบบแนบรูปถ่าย (Upload to Google Drive)
- 🚀 **Priority Badge:** ระบบคัดกรองความเร่งด่วน (🚩 ด่วน - ติดตามพิเศษ / 🏳️ ปกติ - ตามกรอบเวลา)
- 📊 **Admin Worklist:** หน้าจัดการสำหรับเจ้าหน้าที่ สามารถคัดแยกรายการ, เปลี่ยนสถานะ และสรุปจำนวนเป็นรายแผนกได้ทันที
- 🖨️ **Automated Worksheet:** สร้างใบเตรียมจัดซื้อ (Procurement Worksheet) สำหรับพิมพ์หรือบันทึกเป็น PDF พร้อมเว้นช่องเซ็นชื่อเรียบร้อย
- 🔍 **Real-time Tracking:** ผู้ใช้สามารถติดตามสถานะของแต่ละรายการได้ทันทีเมื่อมีการอัปเดตจากหลังบ้าน

## 🛠️ เทคโนโลยีที่ใช้
- **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES6)
- **Backend:** Google Apps Script (GAS)
- **Database:** Google Sheets
- **Storage:** Google Drive API (สำหรับรูปภาพประกอบ)

## 🏗️ วิธีการใช้งานเบื้องต้น
1. เชื่อมต่อระบบกับ Google Sheets ผ่าน URL ใน `frontend/config.js`
2. ตั้งค่าคอลัมน์ใน Sheet ให้ตรงตามหัวข้อใน `backend/Code.gs`
3. ใช้งานผ่านเบราว์เซอร์ได้ทันทีโดยไม่ต้องติดตั้งเพิ่มเติม
